## VitePress

> 一个极简静态网站生成器

- [vitePress](https://github.com/vuejs/vitepress)

基于 Vue 3.x 构建的快速的文档方案。
打包文件体积更小，速度更快。

特性：

- 闪电般快速的冷服务器启
- 动即时热模块更换（HMR）
- 真正的按需编译
